## DISPLAY RULES (IMPORTANT)

### Rule 1 — Carb Range Display
For these categories ONLY: نشويات، فواكه، وجبات عربية
Show: **المتوسط** as the main number
Show below it: `(من X إلى Y جم حسب التحضير)`
Example:
- قرصان → 30 جم (من 25 إلى 35 جم حسب التحضير)
- كبسة دجاج → 15 جم (من 12 إلى 18 جم حسب التحضير)

### Rule 2 — Supermarket Products
Show by fixed unit only: حبة / علبة / كوب / كيس / ملعقة / قطعة / ظرف
No weight input needed for supermarket items.

### Rule 3 — Warning Message
On ANY quantity input screen, show this below the result:
⚠️ القيم تقديرية وقد تختلف حسب التحضير وحجم الحصة

### Rule 4 — Search Aliases
Each food item has search_aliases — use them so searching "رز" or "أرز" both find "رز أبيض مطبوخ"

---

## CALCULATION METHOD (for weight-based items)
carbs = Math.round((weight_in_grams / 100) * carbs_per_100g)
Example: حمص (27g per 100g) → user enters 50g → (50/100)*27 = 13.5 ≈ 14g
